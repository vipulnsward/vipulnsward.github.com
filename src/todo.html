Accounts Manager

You need to create a simple accounts manager{Financial Accounts, I mean}
Your App should be built in Ruby on Rails {3.1, Ruby 1.9.2+},using any type of database you
want, on any Linux Distro of your choice.

Following Tasks need to be completed while you're at it.
  • Create and Manage Users
  • Create and Manage Users Finances
  • User should be able to provide previous balance, create new assets, add salary
    ,add new expenses, along with details of expenses.
  • Separate Reports should project User's Expenses, Income, and Balance Sheet
  • User should be able to update or correct previous entries
  • {Wishlist}Provide Timeline based expences/income sheet, per month or week basis

While you complete the task, you should
  • Create unit tests {Make sure they pass well}
  • Provide proper validations{from rails} and make use of flashes accordingly
  • Provide Documentation for your work, in the code itself
  • Make full use of Rails' MVC elements, partials, etc.
  • Host your project at your github handle
  • Make use of rvm{Optional}
  • {Wishlist}Provide Github-pages using jekyll for your project


Assume any other information as per requirements.
